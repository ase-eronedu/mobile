크롬 사파리 삼성 네이버 카카오톡

크롬(Chrome) 브라우저

  1. 이라온에듀(eraonedu.com) 접속
  2. 화면 우측 상단에 있는 더보기(세로 점 3개 ⋮) 아이콘을 선택합니다.
  3. 메뉴 목록에서 ’홈 화면에 추가’를 선택합니다.
  4. 팝업창이 뜨면 우측 하단의 추가 버튼을 누르면 바탕화면에 아이콘이 생성됩니다.

사파리(Safari) 브라우저

  1. 이라온에듀(eraonedu.com) 접속
  2. 화면 하단에 있는 더보기(가로 점 3개 ⋯) 아이콘을 선택합니다.
  3. 메뉴 창을 위로 살짝 올려 ’홈 화면에 추가(+)’를 찾아 선택합니다.
  4. 우측 상단의 추가 버튼을 누르면 바탕화면에 앱 아이콘이 생성됩니다.

삼성(Samsung) 브라우저

  1. 이라온에듀(eraonedu.com) 접속
  2. 화면 우측 하단에 있는 메뉴(세로 점 3개 ⋮) 아이콘을 선택합니다.
  3. 메뉴 목록에서 ’현재 페이지 추가(+)’를 선택합니다.
  4. 위치 선택 창이 나타나면 ’홈 화면’을 선택합니다.
  5. 마지막으로 추가 버튼을 누르면 완료됩니다.

네이버(NAVER)앱

  1. 검색창 ‘이라온에듀’ 검색 후 접속
  2. 화면 우측 하단에 있는 메뉴(≡ 두 줄) 또는 더보기(··· 점 세개) 아이콘을 터치합니다.
  3. 위치 선택 창이 나타나면 ’홈 화면에 추가’를 선택합니다.
  4. 마지막으로 추가 버튼을 누르면 완료됩니다.

카카오톡(KakaoTalk) 앱

💡
왜 안 되나요?

카카오톡 화면에서는 직접 홈 화면 추가가 안 됩니다. 아래 순서에 따라 반드시 외부 기본 브라우저로 먼저 전환해 주셔야 합니다.

안드로이드 스마트폰 (갤럭시 등)

  1. 카카오톡에서 링크를 눌러 화면을 엽니다.
  2. 화면 우측 하단에 있는 더보기(⋮)를 터치합니다.
  3. 메뉴 창에서 ’다른 브라우저로 열기’를 선택합니다.
  4. 새 브라우저로 전환되면 위 가이드(크롬, 삼성, 사파리)대로 추가합니다.

아이폰 (iOS)

  1. 카카오톡에서 링크를 눌러 화면을 엽니다.
  2. 화면 우측 하단에 있는 나침반(🧭) 아이콘을 터치합니다.
  3. 사파리(Safari)로 창이 열리면 위 가이드(사파리)대로 추가합니다.
